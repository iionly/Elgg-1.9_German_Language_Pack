This Language Pack contains the German language files for Elgg 1.9.X (suitable release for "X" version of Elgg see Changelog) including the bundled plugins.

To install the language pack plugin simply copy the german_language_pack folder into the mod folder of your site and enable the plugin like any other plugin in the admin area of your site. If you had installed a previous release of this plugin, it's best to remove the language files that came with this release from the corresponding folders (or simply delete them together with the other Elgg files when upgrading your site to the latest release anyway). If there are any language files remaining in other plugin folders you should place this language pack plugin below the Elgg bundled plugins in your plugin list in the admin area to override these other language files.

The language used on your Elgg installation can either be set for the whole site via the admin area ("Configure" - "Settings" - "Basic Settings", or in German "Konfigurieren" - "Einstellungen" - "Grundeinstellungen" respectively) or each user can set the language individually on the user account's "Settings" page ("Einstellungen") via the Topbar link.

----------

Dieses Sprachpaket enthält die deutschen Sprachdateien für Elgg 1.9.X (passende Version für jeweiligen "X"-Release von Elgg siehe Changelog) inklusive der mitgelieferten Plugins.

Zum Installieren dieses Sprach-Plugins einfach den german_language_pack-Ordner in das mod-Verzeichnis Deiner Seite kopieren und das Plugin wie ein normales anderes Plugin im Adminbereich Deiner Seite aktivieren. Falls Du bereits einen älteren Release dieses Plugins installiert hattest, solltest Du die Sprachdateien, die in diesem Release enthalten waren, aus den entsprechenden Ordnern löschen (oder einfach mitsamt den anderen Dateien löschen, falls Du sowieso gerade auf die neueste Version updatest). Falls noch andere deutsche Sprachdateien in den Ordnern von anderen Plugins verblieben sind, solltest Du dieses Spach-Plugin unterhalb der bei Elgg mitgelieferten Plugins in der Pluginliste im Adminbereich verschieben, damit diese anderen Sprachdateien unberücksichtigt bleiben.

Die Festlegung der verwendeten Sprache kann entweder global für die gesamte Elgg-Installation über den Admin-Bereich ("Configure" - "Settings" - "Basic Settings" bzw. auf deutsch "Konfigurieren" - "Einstellungen" - "Grundeinstellungen") vorgenommen werden oder jeder Benutzer kann individuell die zu verwendende Sprache auf seiner Settings-Seite ("Einstellungen") via den Link in der Topbar festlegen.

----------

What's the sense of this language pack when a German translation is now included in Elgg 1.9 core?

Actually, I am providing the German translations in Elgg core, too. :-) The continuation of this language pack plugin that provides the German translations is first of all a consequence of my approach when creating / upgrading the German language files. This plugin is (and will be) the basis for any translations I will provide for Elgg core. It's much easier for me to do it this way to not lose track of the current state of both the original English language files and the corresponding German translations. The German language pack plugin will be updated regularly as soon as possible after any new release of Elgg core. I'll try to keep the German translations included in Elgg core as current as possible, too. But it might happen (mainly due to the currentness of translations not being a blocker for new Elgg releases) that the translations in Elgg core might be one release behind (but hopefully never more...). It's for you to decide if the German Language Pack plugin will still be useful for your site or if you can rely on the translations bundled with Elgg core alone.

Was ist der Sinn dieses Sprachpakets wenn nun die deutschen Übersetzungen auch in Elgg 1.9-Core enthalten sind?

Erstmal der Hinweis: die deutschen Übersetzungen in Elgg core sind auch von mir. :-) Die Weiterführung dieses Sprachpaket-Plugins mit den deutschen Übersetzungen ist in erster Linie eine Folge meiner Arbeitsweise beim Erstellen und Aktualisieren der Sprachdateien. Dieses Plugin ist (und wird es weiterhin bleiben) die Grundlage aller Übersetzungen, die ich für Elgg-Core bereitstellen werde. Es ist für mich auf diese Weise deutlich einfacher, den Überblick über Änderungen an den englischen Sprachdateien und den daraus notwendig werdenden Aktualisierungen an den deutschen Sprachdateien zu behalten. Ich werde das German Language Pack-Plugin weiterhin so schnell es mit möglich ist nach Veröffentlichungen neuer Elgg-Versionen aktualisieren. Bei den mit Elgg-Core mitgelieferten Übersetzungen kann es allerdings passieren (hauptsächlich deshalb weil die Aktualität von Sprachdateien auf das Veröffentlichen von neuen Elgg-Versionen keinen Einfluss haben wird), dass die Übersetzungen eine Version hinterher hinken werden (aber hoffentlich nicht mehr...). Ob die Nutzung des German Language Pack-Plugins für Eure Seiten weiterhin einen Sinn machen oder ob die deutschen Übersetzungen in Elgg-Core für Euch ausreichend sind, werdet Ihr allerdings selbst entscheiden müssen.
